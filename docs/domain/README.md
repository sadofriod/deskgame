## Flow Summary

- 房主创建房间并维护房间配置。
- 所有玩家在大厅完成加入与准备后，系统自动进入身份选择阶段。
- 玩家各自从 3 个候选角色中锁定 1 个身份。
- 正式对局共 8 回合，每回合依次经历押牌、玩家行动、伤害结算、发言投票。
- 任一阵营满足胜利条件时立即终局，否则在第 8 回合结束后进行最终判定并输出复盘。
