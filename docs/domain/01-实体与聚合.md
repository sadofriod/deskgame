# 01 实体与聚合

## 1. Room 聚合

**聚合根**：Room

**核心属性**
- roomId
- ownerOpenId
- gameState: wait | start | end
- playerCount
- roleConfig
- currentFloor
- currentStage
- envDeck: 环境牌序列

**不变量**
- playerCount 必须在 5-10。
- gameState = start 时禁止新玩家加入。
- currentFloor 范围为 1-8（终局可标记为 9）。
- currentStage 只能按既定阶段顺序推进。

## 2. Player 实体

**核心属性**
- openId
- nickname
- avatar
- role
- hp（默认 4）
- votePower（默认 1）
- isAlive
- actionCard
- voteTarget
- isReady
- joinTime

**不变量**
- hp <= 0 必须 isAlive = false。
- isAlive = false 时不可提交行动牌或投票。
- actionCard 为空时不可投票和发言。

## 3. Match 聚合

**聚合根**：Match

**核心职责**
- 记录回合列表与阶段推进轨迹。
- 产出复盘可读结构。

**核心属性**
- matchId
- roomId
- rounds
- winnerResult

## 4. Round 值对象

**核心属性**
- floor
- environmentCard
- actionSubmissions
- voteSubmissions
- settlementResult
- stageSnapshots
