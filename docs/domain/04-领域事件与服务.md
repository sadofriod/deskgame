## 2. 领域服务

### 2.1 DealService
- 输入：玩家列表、身份分布、角色池、随机种子
- 输出：玩家身份、角色候选、初始手牌与待确认角色状态
- 约束：结果可复验（同种子同输入同结果）

### 2.2 StageFlowService
- 输入：当前阶段、超时状态、房主指令、回合状态、角色确认状态
- 输出：下一阶段与阶段事件

### 2.3 EnvironmentDeckService
- 输入：环境牌模板、随机种子
- 输出：8 层环境牌序列与未揭示牌信息

### 2.4 SettlementService
- 输入：environmentCard、按顺序执行的行动牌提交、玩家当前状态、当前回合
- 输出：伤害结果、检视结果、玩家状态变化、出局名单

### 2.5 WinnerJudgementService
- 输入：存活玩家身份分布、当前楼层、已完成有屁回合数
- 输出：是否终局、获胜阵营
- 约束：活跃人数 = 0 视为屁者获胜，而非平局
