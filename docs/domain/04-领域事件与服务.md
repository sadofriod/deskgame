- RoundSettled
- PlayerEliminated
- VoteSubmitted
- VoteResolved
- WinnerDecided

## 2. 领域服务

### 2.1 DealService
- 输入：玩家列表、roleConfig、随机种子
- 输出：玩家-身份映射
- 约束：结果可复验（同种子同输入同结果）

### 2.2 StageFlowService
- 输入：当前阶段、超时状态、房主指令
- 输出：下一阶段与阶段事件

### 2.3 EnvironmentDeckService
- 输入：环境牌配置、随机种子
- 输出：8 层环境牌序列

### 2.4 SettlementService
- 输入：environmentCard、行动牌提交、玩家当前状态
- 输出：伤害结果、票权变化、出局名单

### 2.5 WinnerJudgementService
- 输入：存活玩家角色分布、当前楼层
- 输出：是否终局、获胜阵营
