# DeskGame Backend 接入文档

## 概述

DeskGame 后端基于 Node.js + Express + Socket.io 构建，提供 HTTP REST API 和 WebSocket 实时通信。

## HTTP API

### 房间管理

- `POST /rooms` - 创建房间
- `POST /rooms/:roomId/players` - 加入房间
- `DELETE /rooms/:roomId/players/:openId` - 离开房间
- `GET /rooms/:roomId` - 获取房间信息
- `POST /rooms/:roomId/start` - 开始游戏
- `POST /rooms/:roomId/role-selection` - 确认角色
- `POST /rooms/:roomId/actions` - 提交行动
- `POST /rooms/:roomId/environment/reveal` - 揭示环境牌
- `POST /rooms/:roomId/votes` - 提交投票
- `POST /rooms/:roomId/stage/advance` - 推进阶段

### 用户管理

- `POST /users` - 创建/更新用户
- `GET /users/:openId` - 获取用户信息
- `PUT /users/:openId` - 更新用户信息

## WebSocket 命令

- CreateRoom
- JoinRoom
- LeaveRoom
- StartGame
- ConfirmRoleSelection
- SubmitAction
- RevealEnvironment
